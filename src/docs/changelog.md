# 更新日志

记录 LayerAndTreeD 各版本的功能变化。

## v1.0.0

首个正式版本。

### 新增

- OpenLayers 地图引擎集成
- Three.js 三维渲染引擎集成
- 工业数字孪生首页
- 文档中心页面

### 优化

- 粒子背景动画性能优化
- 深色科技风设计语言统一

- 修复地图控件样式冲突
- 修复滚动条样式在部分浏览器失效的问题
