# 常见问题

汇总使用过程中的高频问题与解决方案。

## 安装与构建

### 依赖安装失败怎么办？

请确认 Node 版本不低于 18，删除 `node_modules` 与锁文件后重新安装：

```bash
rm -rf node_modules
bun install
```

### 构建时报类型错误？

项目启用了严格的 TypeScript 检查。请根据 `vue-tsc` 输出逐项修复类型问题。

## 渲染问题

### 三维场景黑屏？

通常是 WebGL 上下文创建失败，请检查：

- 浏览器是否支持 WebGL 2.0
- 显卡驱动是否正常
- 是否存在多个未释放的渲染上下文

### 地图瓦片加载缓慢？

建议启用瓦片缓存并使用 CDN 加速底图资源。

## 性能优化

### 页面卡顿如何排查？

- 使用浏览器性能面板定位长任务
- 对长列表启用虚拟滚动
- 对高频事件使用防抖 / 节流
